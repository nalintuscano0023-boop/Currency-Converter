⚡ RUN THE APPLICATION

🛠️ TROUBLESHOOTING

Issue: Errors while opening directly
Solution: Use local server instead of file mode.

Issue: API not loading
Solution: Check internet connection and reload.

Issue: Voice not working
Solution: Enable AI Speak and check system volume.

Issue: Preferences not saved
Solution: Ensure browser allows local storage.
