💱 AI Currency Converter — Smart Exchange Assistant

A modern AI-powered Currency Converter Web Application built using HTML, CSS, and JavaScript, designed for fast performance, intelligent analysis, and an engaging user experience.
This project combines real-time currency conversion, AI insights, multilingual voice assistance, and premium UI design into a single powerful web app.



 🚀 Features

 ⚡ Performance & Optimization

 Lightning-fast app initialization
 Smart API communication with caching support
 Reduced unnecessary API requests
 Smooth real-time conversion while typing

 🤖 AI Powered Experience

 AI trend analysis for exchange rate behavior
 Smart prediction insights based on rate movement
 Automated conversion announcements using AI voice assistant

 🎙️ AI Voice Assistant

 English (Male) and Hindi (Female) voice support
 Automatic voice fallback detection
 Speech synthesis integration for conversion results and analysis

 🌍 Language & Accessibility

 Instant language switching without page refresh
 English and Hindi interface support
 Accessibility-friendly animations and interaction design

 💾 Personalization

 Recent conversion history tracking
 Favorite currency pair management
 Persistent user preferences using local storage

 🎨 Modern UI / UX

 Glassmorphism inspired interface
 Dark Mode and Light Mode toggle
 Animated gradient background
 Smooth micro-interactions and transitions
 Fully responsive mobile-friendly layout


 🛠️ Tech Stack

 HTML5
 CSS3 (Glassmorphism + Responsive Design)
 Vanilla JavaScript (ES6+)
 Speech Synthesis API
 Exchange Rate API
 Chart.js (Rate visualization)

⚙️ Setup & Installation

1️⃣ Clone Repository


git clone https://github.com/your-username/ai-currency-converter.git


2️⃣ Navigate to Project


cd ai-currency-converter


3️⃣ Start Local Server

Double click:


start-server.bat


or manually:

python -m http.server 8000


Open browser:

http://localhost:8000
